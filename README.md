# SkillSystem

A third-person 3D game template built with Godot 4.7.

## Features

- Third-person character controller (CharacterBody3D)
- Phantom Camera v0.11 (Cinemachine-style camera system)
- Animation blend space — Idle/Walk/Sprint via BlendSpace1D, plus Jump/Jump_Start/Jump_Land
- WASD movement, mouse look, scroll wheel zoom, Shift sprint
- Auto-rotation toward movement direction
- Jolt Physics 3D

## Controls

| Key | Action |
|---|---|
| WASD | Move |
| Shift | Sprint |
| Space | Jump |
| Mouse | Look |
| Scroll wheel | Zoom |

## Getting Started

Open the project in Godot 4.7 and run `world.tscn`.
