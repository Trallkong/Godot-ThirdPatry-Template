# SkillSystem

A third-person 3D game template built with Godot 4.7.

## Features

- Third-person character controller (CharacterBody3D)
- Phantom Camera v0.11 (Cinemachine-style camera system)
- Animation state machine (Idle / Walk / Jump / Jump_Start / Jump_Land)
- WASD movement, mouse look, scroll wheel zoom
- Jolt Physics 3D

## Getting Started

Open the project in Godot 4.7 and run `world.tscn`.
