# AGENTS.md

## Project

Godot 4.7 third-person 3D template (`project.godot`). Remote: `https://github.com/Trallkong/Godot-ThirdPatry-Template.git`.

## Engine Config (non-default)

| Setting | Value |
|---|---|
| renderer | Forward Plus |
| driver (Windows) | d3d12 |
| physics (3D) | Jolt Physics |
| stretch mode | `canvas_items` + `expand` |
| MSAA 3D | disabled (value `1` in `project.godot`) |

## Autoloads

| Name | Path |
|---|---|
| `GlobalVariable` | `gameplay/global_variable/global_variable.gd` — exposes `current_player` |
| `PhantomCameraManager` | from `addons/phantom_camera/` — required by Phantom Camera addon |

## Architecture

| Path | Role |
|---|---|
| `world.tscn` | Main scene (environment, ground, lighting, player, camera) |
| `gameplay/player/` | `Player` (CharacterBody3D), `CharacterSkin` (Node3D, anim state machine) |
| `gameplay/camera/` | `CameraController` — third-person camera via `PhantomCamera3D` |
| `addons/phantom_camera/` | Phantom Camera v0.11 (Cinemachine-like camera system) |
| `models/` | Test GLB model and its PNG texture |
| `gameplay/models/` | Duplicate of `models/` (same files) |

## Animation

`character_skin.tscn` instances `test_model.glb`, which provides the `AnimationPlayer` with clips (Idle, Walk, Jump, etc.). An `AnimationTree` child drives the state machine via `CharacterSkin.speed` (>= 3 → Walk) and `is_jumping`.

## Collision Layers

`3d_physics/layer_1` = "Wall". Player uses `collision_layer = 4` (layer 3).

## Controls

- **WASD** — move (input actions: `move_*`)
- **Mouse** — look (pointer captured on `CameraController._ready`)
- **Scroll wheel** — zoom (camera spring length, 1–5 range)
- **Space** — jump

## Working Here

- Godot MCP tool set available (`godot_*`).
- `.godot/` is gitignored — do not commit engine-generated files.
- UID files (`*.uid`) are generated by Godot 4.4+ alongside `.gd`/`.tscn` — commit them.
- `project.godot` is the only config source. No CI, tests, or linters exist.
